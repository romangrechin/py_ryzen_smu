# py_ryzen_smu

### python script for ryzen_smu kernel module
https://gitlab.com/leogx9r/ryzen_smu

```
usage: ryzen_smu [-h] [--disable_oc DISABLE_OC] [--freq FREQ] [--volt VOLT]
                 [--core_freq CORE_FREQ] [--core_id CORE_ID] [--ccx_id CCX_ID]
                 [--ccd_id CCD_ID] [--ppt_limit PPT_LIMIT]
                 [--tdc_limit TDC_LIMIT] [--edc_limit EDC_LIMIT]

optional arguments:
  -h, --help                   show this help message and exit
  --disable_oc DISABLE_OC
  --freq FREQ                  CPU frequency in Mhz
  --volt VOLT                  CPU voltage in mV
  --core_freq CORE_FREQ        core frequency in Mhz
  --core_id CORE_ID
  --ccx_id CCX_ID
  --ccd_id CCD_ID
  --ppt_limit PPT_LIMIT        PPT limit in mW
  --tdc_limit TDC_LIMIT        TDC limit in mA
  --edc_limit EDC_LIMIT        EDC limit in mA
```
